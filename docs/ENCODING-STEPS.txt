┌──────────────────────────────────────────────────────────────────┐
│ STEP 4: HOW TO ADD COLUMNS    │
└──────────────────────────────────────────────────────────────────┘

    1. ACTIVITY COLUMNS:
       Click [Add Activity]
       → Creates "Act 1", "Act 2", etc.
       → You can add name: "Act 1: Quiz on Cells"

    2. PT COLUMNS:
       Click [Add PT]
       → Creates "PT" column
       → Rename to "PT 1", "Lab Work", etc.

    3. EXAM COLUMNS:
       Click [Add Exam]
       → Creates "Exam" column
       → Rename to "Midterm", "Final", etc.


┌──────────────────────────────────────────────────────────────────┐
│ STEP 5: HOW TO ENCODE SCORES              │
└──────────────────────────────────────────────────────────────────┘

    1. Click on score box
    2. Type the score
    3. Press Tab (next) or Enter (down)
    4. Repeat for all students

    TIPS:
    ✓ Encode by column (faster)
    ✓ Use Tab key to move quickly
    ✓ Empty boxes show "MISS" in red
    ✓ Average auto-calculates


┌──────────────────────────────────────────────────────────────────┐
│ STEP 6: ADJUST WEIGHTS              │
└──────────────────────────────────────────────────────────────────┘

    Default:
    Activity: 30% + PT: 50% + Exam: 20% = 100% ✓

    To change:
    1. Click [Update Weights]
    2. Enter new percentages
    3. Must total 100%
    4. Click OK

    Example:
    Activity: 40% + PT: 40% + Exam: 20% = 100% ✓


┌──────────────────────────────────────────────────────────────────┐
│ STEP 7: SAVE & SEND                     │
└──────────────────────────────────────────────────────────────────┘

    AUTO-SAVE:
    ✓ Every 30 seconds
    ✓ When you close tab
    ✓ Saves to Firebase (cloud) + localStorage (backup)

    MANUAL SEND:
    Click [Send to Registrar]
    → Registrar receives notification
    → Grades saved to cloud immediately


┌──────────────────────────────────────────────────────────────────┐
│ COLOR CODING                                    │
└──────────────────────────────────────────────────────────────────┘

    🔴 RED BOX 
       → Missing score 
       → Shows "MISS"

    🟢 GREEN AVERAGE 
       → 75% or above
       → PASSING ✓

    🔴 RED AVERAGE 
       → Below 75%
       → FAILING ✗


┌──────────────────────────────────────────────────────────────────┐
│ KEYBOARD SHORTCUTS                                               │
└──────────────────────────────────────────────────────────────────┘

    Tab          → Next cell (right)
    Shift + Tab  → Previous cell (left)
    Enter        → Cell below (down)
    Ctrl + S     → Manual save

┌──────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                  │
└──────────────────────────────────────────────────────────────────┘

    Problem: Grades not saving
    Solution:
    1. Check internet connection
    2. Press F12 (open console)
    3. Look for errors
    4. Run firebase-test.html

    Problem: Can't login
    Solution:
    1. Check email is authorized
    2. Password: CCCS2026
    3. Select correct role (Teacher)

    Problem: Firebase error
    Solution:
    1. Open firebase-test.html
    2. Click "Run Tests"
    3. Check which test fails
